CLIO — Platform Summary (Updated March 2026)

Overview
- Clio is a modern academic integrity platform designed to verify authorship, protect student work, and enable confident writing. It uses a "Proof of Write" mechanism by tracking typing behavior (WPM, paste events, pacing) to generate an integrity score. This ensures content is authentically written by the student while giving instructors unmatched visibility into authorship behaviors.

Problem It Solves
- Traditional editors (MS Word, Google Docs, Notion) focus on content creation but don’t surface integrity signals during writing. This leaves teachers relying on reactive, often inaccurate AI-checkers after the fact. Clio fills this gap by acting as a proactive, monitored environment that discourages copy-paste misuse and supports fair assessment, while still delivering a professional writing experience.

Key Features
- Rich Student Editor: Headings, lists, bold/italic, Cloud Autosave (drafts).
- Live Integrity Signals: Real-time WPM tracking, paste detection, and an integrity status chip.
- Session Replay: Periodic snapshots of the essay are taken, allowing teachers to play back the entire writing session like a video.
- Structured Classes & Assignments: Teachers can create classes, enroll students, and assign essays with strict due dates and word count limits.
- Teacher Dashboard: 
  - Grouped submission viewer with detailed integrity reports.
  - Similarity Checking: Compares student submissions against each other to identify collusion.
  - Grading Panel: Assign scores (0-100) and provide rich feedback.
  - Analytics: View class-wide trends and integrity score distributions.
- Exports: Download DOCX or PDF directly from the browser. Export grades as CSV.
- Optional Spotify Integration: Connect an account to control playback and support focus without leaving the writing context.

How It Works (High-Level)
- Frontend (Next.js): The student editor captures text and calculates behavioral signals purely on the client side. Periodic HTML snapshots are taken in the background.
- Backend (Supabase): When submitted, the essay content, metrics (WPM/Paste Count), and replay snapshots are securely sent to the Next.js API. The API uses a Service Role Key to securely interact with the Supabase PostgreSQL database, ensuring students cannot bypass security rules.
- Teacher View: Instructors log in to view aggregated dashboard data, grade submissions, and review flags.

Why It’s Different 
- Integrity-aware writing: Clio surfaces paste detection in real-time to the student, encouraging original thought.
- Proof of Write > Plagiarism Catcher: Because Clio records the drafting process and allows playback, it proves a student *did* write an essay, protecting them from false AI accusations.
- Secure Monolithic Architecture: Built on Next.js 15 and Supabase with strict JWT-based Role-Based Access Control and tight PostgreSQL Row Level Security.

Privacy & Ethics
- Only lightweight behavioral metrics (paste events, WPM) are used for indicators.
- Avoids collecting keystroke-by-keystroke data or using invasive webcam proctoring.
- Intended to encourage honest writing and protect students, not to punish; signals help instructors focus their review realistically.

Tech Stack
- Full-Stack Framework: Next.js 15 (React 19, App Router)
- Database: Supabase (PostgreSQL)
- Styling: TailwindCSS v4 + Shadcn/UI, Glassmorphism design, WebGL Shaders (Dark theme default)
- Editor: Tiptap (ProseMirror-based)
- Security: DOMPurify (XSS protection), bcryptjs (password hashing), Strict RBAC.

Core Value
- Clio promotes honest writing by providing students with a focused, premium workspace, while delivering undeniable "Proof of Write" to educators so they can grade with absolute confidence.
